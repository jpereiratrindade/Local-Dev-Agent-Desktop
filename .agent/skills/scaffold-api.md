# skill: scaffold-api

## Quando usar
- pedido para criar API, backend ou servico HTTP
- pedido para estruturar modulos, rotas, handlers ou camadas

## Objetivo
Montar a espinha dorsal de uma API com arquivos reais, estrutura coerente e configuracao inicial.

## Fluxo
1. Descubra stack e dependencias ja presentes.
2. Defina uma estrutura minima: entrada, rotas, handlers, servicos, modelos, configuracao.
3. Crie pastas e arquivos essenciais.
4. Preencha exemplos minimos funcionais.
5. Valide build ou comando de execucao.

## Regras
- Nao invente frameworks fora do padrao do projeto sem justificativa.
- Prefira endpoint simples de healthcheck e um modulo real pequeno antes de escalar.
- Documente como subir e testar localmente.
