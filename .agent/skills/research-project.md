# skill: research-project

## Quando usar
- pedido para elaborar projeto de pesquisa
- pedido para formular problema, objetivos, metodologia ou corpus
- pedido para estruturar justificativa, hipoteses e entregaveis

## Funcao
Guiar o desenho de um projeto de pesquisa com rigor, sem sufocar invencao conceitual.

## Fluxo sugerido
1. Identifique tema, problema, contexto e material disponivel.
2. Estruture questao central, justificativa e objetivos.
3. Delineie hipoteses, corpus, metodo e estrategia analitica.
4. Aponte referencias, lacunas de evidencia e proximos aprofundamentos.
5. Quando fizer sentido, produza um esqueleto de projeto ou uma secao completa em prosa.

## Flexibilidade
- Esta skill organiza o pensamento, nao substitui elaboracao intelectual.
- Adapte a estrutura ao campo disciplinar e ao estado real do projeto.
- Combine com RAG, biblioteca local, web aprovada ou MCP quando isso ampliar a base empirica.
