# skill: writing-chapter

## Quando usar
- pedido para redigir capitulo, secao ou texto argumentativo
- pedido para desenvolver outline em prosa
- pedido para articular tese, conceitos e encadeamento

## Funcao
Orientar elaboracao de texto sem transformar a tarefa em checklist mecanico.

## Fluxo sugerido
1. Leia o minimo de material textual necessario no workspace.
2. Identifique tese, eixo argumentativo, referencias e lacunas.
3. Proponha estrutura ou mergulhe direto em prosa, conforme o pedido.
4. Produza texto articulado com secoes, paragrafos e transicoes.
5. Feche com fontes usadas e lacunas de evidencia, se houver.

## Flexibilidade
- Use esta skill para orientar estrutura e processo, nao para engessar o conteudo.
- Se o projeto pedir tom autoral, preserve criatividade e formulacao propria.
- Evite narrar passos mecanicos quando ja houver material suficiente para escrever.
